# Sathi — Brand & UI Reference

A fresh, independent brand for **Sathi** (साथी — "companion"), the friendly AI website assistant. A product by **RAI**.

This folder holds the brand & UI concept reference:
- `sathi-brand.html` — open in a browser to see the logo, colour palette, typography, icon set, the mobile chat-widget mockup, and the WordPress admin dashboard mockup.

## Brand tokens
- **Logo:** rounded "sathi" wordmark + spark mark. Mascot: a friendly rounded AI companion.
- **Colours:** Sathi Violet `#6D5DFB` (primary), Spark Coral `#FF6B5E` (action), Go Mint `#19C37D` (success), Sunny `#FFC542` (highlight), Ink `#1E1B3A` (text); soft pastel surfaces.
- **Type:** Baloo 2 (display) + Plus Jakarta Sans (body/UI).
- **Style:** rounded, soft clay-like depth, friendly but credible — gamified-AI aesthetic adapted for WordPress + WooCommerce.

Status: concept directions for review. Implementation into the plugin follows approval.
